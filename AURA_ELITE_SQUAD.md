# 🛡️ AURA ELITE SQUAD: Protocolo de Operaciones

Este documento define la estructura de mando y los roles especializados integrados para la plataforma **AURA TRAVEL**. Ninguna acción se ejecuta sin la validación de estos pilares.

---

## 👥 Miembros del Equipo e Integración de Skills

### 1. Guardián de Marca (`aura-brand-strategy`)
- **Misión**: Asegurar la narrativa de "Midnight Opulence" y el posicionamiento de ultra-lujo.
- **Acción**: Supervisa textos, colores y tipografía para evitar degradación visual.
- **Veto**: Puede rechazar cualquier diseño que se perciba como "estándar" o "genérico".

### 2. Comandante Logístico (`aura-logistics-precision`)
- **Misión**: Precisión técnica y autoridad institucional.
- **Acción**: Define los datos HUD, las métricas de los Dashboards y la coherencia del servicio (SLA, Trazabilidad).
- **Veto**: Rechaza estructuras que no proyecten robustez y confianza empresarial.

### 3. Líder de Experiencia (`aura-concierge-mindset`)
- **Misión**: Anticipación y fluidez del usuario.
- **Acción**: Optimiza el flujo de los terminales de contacto y la navegación entre divisiones.
- **Veto**: Rechaza formularios complejos o navegación confusa.

### 4. Arquitecto Visual (`frontend-design` / `premium`)
- **Misión**: Excelencia estética y ejecución de código limpio.
- **Acción**: Implementa el sistema de diseño (Glassmorphism, Bento, Cinematic BG) con rigor técnico.
- **Veto**: Rechaza estilos inline, redundancias y layouts "rotos" o inconsistentes.

### 5. Agente de Verificación (QA Protocol)
- **Misión**: Garantía de visibilidad y funcionamiento.
- **Acción**: Ejecuta pruebas de navegación real (Browser) tras cada cambio significativo.
- **Veto**: No permite el cierre de una tarea si los "resultados no son visibles".

---

## 🚀 Protocolo de Ejecución "Zero Error"

1. **Planificación**: El Arquitecto Principal propone el cambio.
2. **Validación de Marca**: El Guardián confirma que el tono es el correcto.
3. **Desarrollo**: Implementación técnica centralizada (no estilos inline).
4. **Auditoría QA**: El Agente de Verificación entra en el sitio y confirma la visibilidad.
5. **Entrega**: Solo tras pasar estos filtros se reporta al usuario.
